# silentorb-workbench

Development workspace for **Marloth Story** and **Tome**.

- `marloth-story/` — Marloth design graph (`content/`, domain docs, migrations)
- `tome/` — Tome packages (`tome-db`, `tome-editor`, `tome-static-site`) and tooling docs

See [`AGENTS.md`](./AGENTS.md) for agent and developer conventions.
